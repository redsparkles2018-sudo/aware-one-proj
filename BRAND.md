## Logo

### Concept 1 — Primary Wordmark
`AWARE 1`
- "AWARE" in Inter, weight 800 (ExtraBold), slate (#1E2B3C)
- "1" in Inter, weight 300 (Light), teal (#00A99D)
- Letter spacing: -0.02em
- Use this as the main logo on website header, documents, presentations

### Concept 2 — Icon + Wordmark
`A¹` icon block + "Aware One" text
- Icon: 52×52px teal (#00A99D) rounded rectangle (radius 10px)
- "A¹" inside icon: Inter 800, white, with superscript "1"
- Wordmark: "Aware" in Inter 700 slate + "One" in Inter 400 teal
- Sub-label: "Business Consultancy" in 10px uppercase, slate gray
- Use for favicon, profile photos, social media, app icons, business cards

### Usage Rules
- Never stretch or distort the logo
- Minimum clear space: equal to height of the "A" character on all sides
- On dark backgrounds: use white "AWARE" + teal-light (#00C4B7) for the "1"
- On teal backgrounds: use white for all elements
- Never place on busy backgrounds without a solid backing

## Colour Palette

| Name | Hex | Usage |
|------|-----|-------|
| Deep Slate | `#1E2B3C` | Primary — headlines, logo, nav, dark sections |
| Teal | `#00A99D` | Accent — CTAs, highlights, icon backgrounds, links |
| Teal Light | `#00C4B7` | Hover states, logo on dark backgrounds |
| Teal Pale | `#E6F7F6` | Section backgrounds, card tints |
| White | `#FFFFFF` | Page background, cards |
| Slate Gray | `#6B7C93` | Body text, captions, sub-labels |
| Light Gray | `#E8EDF2` | Borders, dividers, input fields |
| Off White | `#F8FAFB` | Alternate section backgrounds |

### Colour Rules
- Primary CTA buttons: Teal (`#00A99D`) background, white text
- Secondary/ghost buttons: Slate border + text
- Headlines: Deep Slate (`#1E2B3C`)
- Body text: Slate Gray (`#6B7C93`)
- Links: Teal, underline on hover
- Dark sections (footer, hero): Deep Slate background, white text

## Typography

### Primary Font: Inter (Google Fonts — free)
```
@import url('https://fonts.googleapis.com/css2?family=Inter:wght@300;400;500;600;700;800;900&display=swap');
```

| Use | Weight | Size (desktop) |
|-----|--------|----------------|
| H1 — Hero headline | 800 | 56–64px |
| H2 — Section title | 700 | 36–40px |
| H3 — Card/sub title | 600 | 24px |
| H4 — Label/small title | 600 | 18px |
| Body | 400 | 16px |
| Small / caption | 400 | 14px |
| Eyebrow / tag | 600, uppercase | 11–12px, 0.1em tracking |

### Letter Spacing
- Headlines: -0.02em (tight)
- Eyebrow labels: +0.1em (wide)
- Body: default (0)

## Voice & Tone

- **Direct** — say what you mean, skip the filler
- **Confident** — you know your stuff, no need to hedge
- **Human** — talking to a business owner, not a committee
- **Avoid:** jargon, buzzwords, corporate speak ("synergy", "leverage", "deep dive")

### Example
❌ "We leverage cutting-edge AI solutions to optimise your operational workflows."
✅ "We help your business run better using the latest AI tools."
